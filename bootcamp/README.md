# 🚀 Bootcamp Interview Prep - LLM Systems Engineer

## Obiettivo
Preparazione intensiva per interview tecnica con focus su:
- **LangGraph** e agenti autonomi
- **Python** avanzato (AsyncIO, FastAPI, SQLAlchemy)
- **RAG** e sistemi di vector search
- **System Design** per AI/LLM
- **Observability** e debugging

---

## 📅 **PROGRAMMA 7 GIORNI**

### **DAY 1**: LangGraph Mastery 🤖
- [ ] **Teoria**: State management, nodes, edges, conditionals
- [ ] **Pratica**: Estendi il tuo agent con nuove funzionalità
- [ ] **Demo**: Crea un agent multi-step interattivo
- [ ] **Questions**: Preparati a spiegare StateGraph design

### **DAY 2**: Python Advanced Patterns 🐍
- [ ] **AsyncIO**: Event loop, coroutines, background tasks
- [ ] **Type Hints**: Generics, Protocol, TypedDict
- [ ] **Design Patterns**: Factory, Observer, State
- [ ] **Demo**: Implementa un pattern complex async

### **DAY 3**: RAG & Vector Search 🔍
- [ ] **Embedding strategies**: Chunking, retrieval optimization
- [ ] **Vector databases**: pgvector vs alternatives
- [ ] **Evaluation**: Metrics, benchmarking RAG quality
- [ ] **Demo**: Advanced RAG pipeline con reranking

### **DAY 4**: FastAPI & API Design 🌐
- [ ] **Advanced FastAPI**: Middleware, dependency injection, lifespan
- [ ] **Error handling**: Custom exceptions, validation
- [ ] **Testing**: pytest-asyncio, httpx, mocking
- [ ] **Demo**: API resilient con circuit breaker

### **DAY 5**: System Design per AI/LLM ⚡
- [ ] **Scalabilità**: Horizontal scaling, load balancing
- [ ] **Reliability**: Circuit breaker, retry, fallback
- [ ] **Observability**: Metrics, tracing, alerting
- [ ] **Demo**: Design di un sistema LLM production-ready

### **DAY 6**: Observability & Debugging 📊
- [ ] **Prometheus**: Custom metrics, PromQL
- [ ] **Logging**: Structured logging, correlation IDs
- [ ] **Debugging**: Distributed tracing, performance
- [ ] **Demo**: Debug un problema complex in produzione

### **DAY 7**: Mock Interview & Review 🎯
- [ ] **System Design**: WhiteBoard exercise
- [ ] **Coding**: Live coding session
- [ ] **Behavioral**: STAR methodology
- [ ] **Project Demo**: Presenta il tuo sistema
